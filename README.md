## 🚀 Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the game:
   ```bash
   python app.py
   ```
4. Open your browser and navigate to `http://localhost:5000`
